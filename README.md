# Java Playground ☕

자바의 핵심 기술들을 직접 실습하고 정리하는 개인 학습용 레포지토리입니다.
기본 문법부터 멀티스레드, 파일 I/O, 소켓 프로그래밍까지 단계적으로 학습합니다.

---

## 📂 구성

| 폴더명 | 학습 주제 | 주요 내용 |
|--------|------------|-----------|
| `core-playground` | 자바 기본 문법 | 제네릭, 람다, 스트림, 익명 클래스 등 |
| `concurrency-playground` | 멀티스레드 & 동기화 | Thread, ExecutorService, synchronized, Atomic 등 |

---

## 🧠 학습 목표

- 자바 언어의 핵심 문법과 동작 원리 이해
- 동시성 프로그래밍에 대한 실습 기반 확보
- 네트워크 및 파일 I/O 구조 체험

---

## 🛠 개발 환경

- **Language:** Java 21+
- **IDE:** IntelliJ IDEA
- **OS:** macOS / Windows
